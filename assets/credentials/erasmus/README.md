# Erasmus+ project photos

Add up to 3 photos per certification using this folder structure:

```
erasmus/
  ai-sustainibility/photo-1.jpg
  ai-sustainibility/photo-2.jpg
  ai-sustainibility/photo-3.jpg
  cyberkindnes/photo-1.jpg
  ...
```

Slugs match each Youthpass title in `assets/js/script.js` (`erasmusCertificates`).
